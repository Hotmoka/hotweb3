# hotweb3
JavaScript/TypeScript client for a remote [Hotmoka](https://www.hotmoka.io) node


## Build 

### Requirements

-   [Node.js](https://nodejs.org)
-   [npm](https://www.npmjs.com/)

### Install dependencies
```bash
npm install
```
### Bundle source code

```bash
npm run bundle
```

## Tests
```bash
npm run test:all
```
